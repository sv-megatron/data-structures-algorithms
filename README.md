# DSA Patterns Practice 🚀

This repository contains my structured practice of Data Structures and Algorithms problems, organized by **problem-solving patterns** rather than just problem numbers.

## 🎯 Goal

* Build strong pattern recognition for coding interviews
* Solve 200+ problems across core DSA topics
* Maintain clean, well-documented solutions

---

## 🧠 Approach

Each problem includes:

* Multiple approaches (brute force → optimized)
* Clear explanation of thought process
* Time & space complexity analysis
* Edge cases and key learnings

---

## 📂 Folder Structure

Problems are grouped by **primary pattern**:

* arrays-hashing
* sliding-window
* two-pointers
* stack / queue
* linked-list
* trees / graphs
* heap
* greedy
* backtracking
* dynamic-programming
* bit-manipulation

---

## 🔍 How to Use This Repo

* Start with a pattern (e.g., Sliding Window)
* Solve problems in increasing difficulty
* Revise using `patterns.md`

---
